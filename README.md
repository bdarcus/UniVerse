## üöÄ Getting Started

### Prerequisites
- An active API Key for **Google Gemini**.

### Installation
1.  **Clone the repository** (or download the source).
2.  **Environment Setup**: Ensure your environment has access to the `process.env.API_KEY` variable.
3.  **Run Development Server**:
    ```bash
    npm install
    npm run dev
    ```
4.  **View the App**: Open [http://localhost:3000](http://localhost:3000) in your browser.

### Faculty Assessment View
To experience the faculty-side grading interface, navigate to the **"Faculty View"** in the sidebar. This allows you to simulate the rubric-based assessment process that feeds directly back into the student's Unified Portfolio.
